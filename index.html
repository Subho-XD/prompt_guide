Best Practices

✅ Do This

  • Start with role definition ("Act as a...")
  • Provide specific context and constraints
  • Request precise output formats
  • Include examples when helpful
  • Use clear, structured language
  • Test and iterate your prompts

❌ Avoid This

  • Vague requests ("improve this", "make it better")
  • Missing context or background
  • Overly complex single requests
  • Assuming AI knows your specific domain
  • Undefined output requirements
  • No feedback or iteration
